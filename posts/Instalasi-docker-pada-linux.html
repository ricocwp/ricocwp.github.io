Instalasi Docker Pada Linux

Pada tutorial kali ini kita akan belajar bagaimana menginstal docker container pada Linux, bagi yang belum mengetahui tentang docker bisa mengunjungi tutorial sebelumnya tentang Pengenalan Docker Container. Pada lab ini, saya menggunakan Ubuntu 16.04.5 LTS (Ubuntu Server) mungkin akan sedikit berbeda cara instalasinya jika bukan dari varian Ubuntu atau base Debian. Oke, langsung saja kita mulai cara untuk instalasi docker container.

Instalasi Docker Pada Ubuntu

Pada kasus ini, kita akan menginstal docker dari repository docker. Docker yang kita install adalah edisi Docker-CE (Docker Engine - Community).

Lakukan update paket terlebih dahulu

$ sudo apt-get update

Instal paket-paket yang dibutuhkan

$ sudo apt-get install \
    apt-transport-https \
    ca-certificates \
    curl \
    gnupg-agent \
    software-properties-common

Tambahkan GPG key docker

$ curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

Tambahkan repository docker

$ sudo add-apt-repository \
    "deb [arch=amd64] https://download.docker.com/linux/ubuntu \
    $(lsb_release -cs) \
    stable"

Lakukan update untuk membaca perubahan pada repository

$ sudo apt-get update

Sekarang instal docker-ce pada sistem operasi kita

$ sudo apt-get install docker-ce docker-ce-cli containerd.io

Jika sudah selesai proses instalasinya, kita cek apakah docker sudah berhasil terpasang atau belum. Kita bisa melakukan pengecekan dengan melihat versi dari docker di sistem operasi kita

Menambahkan User Kedalam Group Docker

Buat group baru dengan nama docker.

$ sudo groupadd docker

Tambahkan user kita kedalam group docker.

$ sudo usermod -aG docker $USER

Untuk masuk kedalam group docker, tinggal masukkan perintah berikut:

$ newgrp docker
